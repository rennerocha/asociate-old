============
Installation
============


Stable release
--------------

To install Association Management, run this command in your terminal:

.. code-block:: console

    $ pip install association_management

This is the preferred method to install Association Management, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Association Management can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/rennerocha/association_management

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/rennerocha/association_management/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/rennerocha/association_management
.. _tarball: https://github.com/rennerocha/association_management/tarball/master
